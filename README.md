# 📢 PENGUMUMAN E-SPORT: MOBILE LEGENDS 🎮🔥

Kompetisi E-Sport Mobile Legend hadir untuk mewadahi minat dan bakat siswa tingkat SMA/SMK/MA sederajat, khususnya bagi siswa aktif **SMK N 1 Sanden** semua jurusan.

Ajang ini bukan sekadar permainan, tetapi menjadi sarana untuk mengasah:
✨ **Strategi & analisis permainan**
✨ **Disiplin & kerja sama tim**

Dengan sistem turnamen yang terstruktur, peserta akan merasakan pengalaman kompetisi layaknya profesional, mulai dari babak penyisihan hingga final di arena utama.
